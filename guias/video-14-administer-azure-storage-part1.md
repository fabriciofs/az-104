## Descricao do Conteudo

Pronto para mergulhar no mundo do Armazenamento do Azure? Neste modulo, voce aprendera como configurar e gerenciar contas do Armazenamento do Azure, proteger endpoints de armazenamento, implementar o Armazenamento de Blobs e o Azure Files. Ao final, voce estara preparado para otimizar solucoes de armazenamento e aprimorar a seguranca de dados. Prepare-se para elevar suas habilidades em nuvem!

## Topicos Abordados

- Configuracao de contas do Armazenamento do Azure (Azure Storage Accounts)
- Gerenciamento de contas de armazenamento
- Protecao de endpoints de armazenamento
- Implementacao do Armazenamento de Blobs (Azure Blob Storage)
- Implementacao do Azure Files
- Otimizacao de solucoes de armazenamento
- Seguranca de dados no armazenamento

## Conceitos-Chave para o Exame

### Azure Storage Accounts

- Tipos de contas de armazenamento (Standard, Premium)
- Redundancia (LRS, ZRS, GRS, RA-GRS, GZRS, RA-GZRS)
- Performance tiers
- Access tiers (Hot, Cool, Archive)

### Azure Blob Storage

- Block blobs, Append blobs, Page blobs
- Containers e organizacao de dados
- Niveis de acesso (Hot, Cool, Archive)
- Lifecycle management

### Azure Files

- Compartilhamentos de arquivos SMB
- Azure File Sync
- Snapshots de compartilhamentos

### Seguranca de Armazenamento

- Storage Account Keys
- Shared Access Signatures (SAS)
- Azure AD authentication
- Encryption at rest e in transit
