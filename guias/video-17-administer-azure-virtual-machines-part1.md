## Descricao do Conteudo

Explore the essentials of managing Azure virtual machines, including configuring availability, determining sizing, and connecting using various protocols. This session provides a comprehensive overview to help you effectively plan, create, and manage Azure VMs.

## Topicos Abordados

- Fundamentos de Azure Virtual Machines
- Configuracao de disponibilidade (Availability)
- Determinacao de dimensionamento (Sizing)
- Conexao usando diversos protocolos
- Planejamento de VMs
- Criacao de VMs
- Gerenciamento de VMs

## Conceitos-Chave para o Exame

### Planejamento de VMs

- Selecao de regiao
- Naming conventions
- Sizing e familias de VMs
- Sistema operacional (Windows/Linux)
- Custos e pricing

### Disponibilidade e Resiliencia

- Availability Sets
- Availability Zones
- Virtual Machine Scale Sets (VMSS)
- Fault domains e Update domains
- SLA de VMs

### Dimensionamento (Sizing)

- Series de VMs (B, D, E, F, G, H, L, M, N)
- General purpose vs Compute optimized
- Memory optimized vs Storage optimized
- GPU VMs
- Resize de VMs

### Conectividade

- RDP (Remote Desktop Protocol) - Windows
- SSH (Secure Shell) - Linux
- Azure Bastion
- Just-in-time (JIT) VM access
- Serial console

### Armazenamento de VMs

- OS disks
- Data disks
- Temporary disks
- Managed disks vs Unmanaged disks
- Disk types (Standard HDD, Standard SSD, Premium SSD, Ultra Disk)
