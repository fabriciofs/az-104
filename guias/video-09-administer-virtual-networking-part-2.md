## Descricao do Conteudo

Welcome back to the world of Azure Virtual Networking! In this second part of the module, you'll continue your exciting journey to master the art of virtual networking in Azure. Imagine being able to create and manage virtual networks, configure robust security with network security groups, and host domains on Azure DNS - all while enhancing your skills and boosting your career. Let's dive in and unlock more of the full potential of Azure Virtual Networking together!

## Topicos Abordados

- Continuacao do modulo de Virtual Networking
- Configuracao avancada de Virtual Networks
- Network Security Groups (NSGs) - configuracao detalhada
- Azure DNS - hospedagem de dominios
- Praticas avancadas de seguranca de rede

## Conceitos-Chave para o Exame

### Network Security Groups (NSGs) - Aprofundamento

- Regras de seguranca inbound e outbound
- Prioridade de regras (numeros menores = maior prioridade)
- Default rules vs custom rules
- Application Security Groups (ASGs)

### Azure DNS

- Zonas DNS publicas e privadas
- Record sets (A, AAAA, CNAME, MX, NS, PTR, SOA, SRV, TXT)
- Alias records para recursos Azure
- Integracao com Virtual Networks

### Service Endpoints

- Conexao segura a servicos Azure
- Extensao da identidade da VNet para servicos
- Configuracao por subnet
