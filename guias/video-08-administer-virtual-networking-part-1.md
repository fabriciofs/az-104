## Descricao do Conteudo

Welcome to the world of Azure Virtual Networking! In this module, you'll embark on an exciting journey to master the art of virtual networking in Azure. Imagine being able to create and manage virtual networks, configure robust security with network security groups, and host domains on Azure DNS - all while enhancing your skills and boosting your career. Let's dive in and unlock the full potential of Azure Virtual Networking together!

## Topicos Abordados

- Introducao ao Azure Virtual Networking
- Criacao e gerenciamento de Virtual Networks (VNets)
- Configuracao de seguranca com Network Security Groups (NSGs)
- Hospedagem de dominios no Azure DNS
- Fundamentos de redes virtuais no Azure

## Conceitos-Chave para o Exame

### Virtual Networks (VNets)

- Redes virtuais isoladas no Azure
- Segmentacao de recursos em subnets
- Espacos de endereco IP privados

### Network Security Groups (NSGs)

- Regras de seguranca de entrada e saida
- Filtragem de trafego de rede
- Associacao com subnets e interfaces de rede

### Azure DNS

- Hospedagem de zonas DNS
- Resolucao de nomes para recursos Azure
- Integracaoo com Virtual Networks
