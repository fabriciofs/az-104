This is the final part of our Azure Virtual Networking module! In Part 3, you'll complete your journey to master virtual networking in Azure. Imagine being able to create and manage virtual networks, configure robust security with network security groups, and host domains on Azure DNS - all while enhancing your skills and boosting your career. - Resolucao de nomes privada dentro de VNets
- Vinculacao de zonas DNS a Virtual Networks
- Auto-registration de VMs

### Azure Firewall

- Firewall gerenciado e baseado em nuvem
- Regras de aplicacao e rede
- Threat intelligence
- Integracao com Virtual Networks

### Network Watcher

- Diagnostico e monitoramento de rede
- IP Flow Verify
- Next Hop
- Connection troubleshoot
- Packet capture

### Private Endpoints

- Conexao privada a servicos Azure PaaS
- Elimina exposicao a internet publica
- Integracao com Private DNS Zones
