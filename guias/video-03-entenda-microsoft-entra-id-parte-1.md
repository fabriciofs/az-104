## Descricao do Conteudo

Este video e a primeira parte do modulo sobre Microsoft Entra ID (anteriormente Azure Active Directory). O conteudo aborda como criar, configurar e gerenciar identidades usando o Microsoft Entra ID.

### O que voce aprendera

- Conceitos fundamentais do Microsoft Entra ID
- Diferencas entre Entra ID e Active Directory on-premises
- Tipos de identidades no Azure
- Conceitos de tenant e subscription
- Gerenciamento basico de usuarios e grupos

---

## Topicos Abordados

### 1. Introducao ao Microsoft Entra ID

| Conceito                 | Descricao                                                |
| ------------------------ | -------------------------------------------------------- |
| **Microsoft Entra ID**   | Servico de gerenciamento de identidades baseado em nuvem |
| **Tenant**               | Instancia dedicada do Entra ID para uma organizacao      |
| **Azure AD vs Entra ID** | Renomeacao do servico (mesmo produto)                    |

### 2. Tipos de Identidades

- **Usuarios** - Identidades humanas
- **Grupos** - Colecoes de usuarios para gerenciamento simplificado
- **Entidades de Servico** - Identidades para aplicacoes
- **Identidades Gerenciadas** - Identidades automaticas para recursos Azure

### 3. Conceitos de Tenant

| Aspecto        | Detalhe                                    |
| -------------- | ------------------------------------------ |
| **Definicao**  | Organizacao representada no Azure          |
| **Relacao**    | Um tenant pode ter multiplas subscriptions |
| **Isolamento** | Cada tenant e completamente isolado        |

### 4. Comparacao: Entra ID vs Active Directory

| Caracteristica  | Active Directory | Microsoft Entra ID |
| --------------- | ---------------- | ------------------ |
| **Localizacao** | On-premises      | Cloud              |
| **Protocolo**   | Kerberos, LDAP   | SAML, OAuth, OIDC  |
| **Estrutura**   | Floresta/Dominio | Tenant plano       |
| **Gestao**      | GPO              | Conditional Access |

---

## Conceitos-Chave para o Exame

1. **Tenant vs Subscription**

   - Tenant: Container de identidade
   - Subscription: Container de billing e recursos

2. **Tipos de Licenca do Entra ID**

   - Free
   - P1
   - P2

3. **Roles Importantes**

   - Global Administrator
   - User Administrator
   - Application Administrator

4. **Autenticacao na Nuvem**
   - Cloud-only identities
   - Synchronized identities
   - Federated identities
