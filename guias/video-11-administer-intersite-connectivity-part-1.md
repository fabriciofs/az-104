## Descricao do Conteudo

Welcome to the Administer Intersite Connectivity module! In this session, you'll dive into the fascinating world of virtual network peering, mastering the art of configuring and managing Azure virtual networks. You'll learn how to control traffic flow with routes and explore the uses of service endpoints and private links. This module is designed to empower you with the skills to enhance network performance and security.

## Topicos Abordados

- Virtual Network Peering
- Configuracao e gerenciamento de redes virtuais Azure
- Controle de fluxo de trafego com rotas
- Service Endpoints
- Private Links

## Conceitos-Chave para o Exame

### Virtual Network Peering

- Conexao de VNets na mesma regiao (Regional Peering)
- Conexao de VNets em regioes diferentes (Global Peering)
- Trafego privado atraves do backbone Microsoft
- Nao transitivo por padrao
- Configuracao bidirecional necessaria

### User Defined Routes (UDR)

- Tabelas de rota personalizadas
- Controle granular do trafego
- Next hop types: Virtual Appliance, VNet Gateway, Internet, None
- Associacao com subnets

### Service Endpoints

- Extensao da identidade VNet para servicos Azure
- Acesso seguro a servicos PaaS
- Rotas otimizadas para servicos Azure
- Configuracao por subnet

### Private Link

- Acesso privado a servicos Azure PaaS
- Conexao via Private Endpoint
- Elimina exposicao a internet publica
- Funciona com DNS privado
