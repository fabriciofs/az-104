## Descrição do Conteúdo

Este é o vídeo de prévia do curso AZ-104: Microsoft Azure Administrator. O vídeo apresenta uma visão geral do que o aluno pode esperar do curso, incluindo os recursos disponíveis, benefícios e estrutura do treinamento.

### O que o Curso Oferece

O curso AZ-104 prepara candidatos para a certificação **Microsoft Certified: Azure Administrator Associate**, cobrindo:

- Gerenciamento de identidades e governança no Azure
- Implementação e gerenciamento de armazenamento
- Implantação e gerenciamento de recursos de computação
- Implementação e gerenciamento de redes virtuais
- Monitoramento e manutenção de recursos do Azure

## Público-Alvo

Este curso é ideal para:

- Administradores de sistemas migrando para a nuvem
- Profissionais de TI que gerenciam infraestrutura Azure
- Candidatos à certificação Azure Administrator
- Profissionais com experiência em AZ-900 buscando o próximo nível

---

## Pré-requisitos Recomendados

Antes de iniciar o AZ-104, é recomendado ter:

1. **Conhecimento básico de Azure** (AZ-900 ou equivalente)
2. Familiaridade com:
   - Sistemas operacionais Windows e Linux
   - Conceitos de rede (TCP/IP, DNS, firewalls)
   - Virtualização
   - PowerShell ou Azure CLI
