## Descricao do Conteudo

Welcome back to the Administer Intersite Connectivity module! In this second part of the session, you'll continue exploring the fascinating world of virtual network peering and further develop your skills in configuring and managing Azure virtual networks. You'll dive deeper into controlling traffic flow with routes and examine advanced uses of service endpoints and private links. This module is designed to empower you with the skills to enhance network performance and security.

## Topicos Abordados

- Continuacao do modulo de Intersite Connectivity
- Virtual Network Peering avancado
- Configuracao avancada de rotas

- Private Links - implementacao detalhada
- VPN Gateway e ExpressRoute

## Conceitos-Chave para o Exame

### VPN Gateway

- Conexao segura entre Azure e on-premises
- Site-to-Site VPN para conexoes permanentes
- Point-to-Site VPN para clientes individuais
- VPN Gateway SKUs e capacidades
- Active-Active e Active-Passive configurations

### ExpressRoute

- Conexao privada dedicada ao Azure
- Nao passa pela internet publica
- Maior largura de banda e menor latencia
- ExpressRoute Global Reach
- ExpressRoute Direct

### Gateway Transit

- Permite que VNets peered usem o mesmo gateway
- Reduz custos e complexidade
- Configuracao em peering settings
- Hub-and-spoke topology

### Network Virtual Appliances (NVA)

- Firewalls de terceiros
- Load balancers
- WAN optimizers
- Integracao com User Defined Routes
