## Descrição do Conteúdo

Este vídeo apresenta a introdução oficial ao curso AZ-104: Azure Administrator. Voltado para profissionais de TI, o curso ensina como gerenciar ambientes Azure de forma completa.

### O que o Curso Ensina

O curso AZ-104 capacita profissionais de TI a:

| Habilidade                             | Descrição                                      |
| -------------------------------------- | ---------------------------------------------- |
| **Gerenciar assinaturas Azure**        | Administrar subscriptions e recursos           |
| **Proteger identidades**               | Implementar segurança de identidade            |
| **Administrar infraestrutura**         | Gerenciar recursos de computação               |
| **Configurar redes virtuais**          | Implementar VNets e conectividade              |
| **Conectar sites Azure e on-premises** | Implementar soluções híbridas                  |
| **Gerenciar tráfego de rede**          | Configurar load balancers e traffic management |
| **Implementar armazenamento**          | Configurar storage solutions                   |
| **Criar e escalar VMs**                | Gerenciar virtual machines e scale sets        |
| **Implementar web apps e containers**  | Configurar App Service e containers            |
| **Backup e compartilhamento**          | Implementar proteção de dados                  |
| **Monitorar soluções**                 | Configurar Azure Monitor e alertas             |

---

## Objetivos do Curso

### Após completar o curso, você será capaz de:

1. **Gerenciar Identidades e Governança**

   - Criar e gerenciar usuários e grupos no Entra ID
   - Implementar RBAC
   - Configurar Azure Policy

2. **Implementar Armazenamento**

   - Criar storage accounts
   - Configurar blob storage e file shares
   - Implementar segurança de armazenamento

3. **Implantar Recursos de Computação**

   - Criar e configurar VMs
   - Implementar ARM templates e Bicep
   - Gerenciar containers

4. **Configurar Redes Virtuais**

   - Criar VNets e subnets
   - Implementar NSGs e ASGs
   - Configurar DNS e load balancing

5. **Monitorar e Manter Recursos**
   - Configurar Azure Monitor
   - Implementar backup e recovery
   - Gerenciar alertas

## Público-Alvo

Este curso é destinado a:

- **Administradores de TI** migrando para a nuvem
- **Engenheiros de infraestrutura** Azure
- **Profissionais de suporte** que gerenciam recursos Azure
- **Candidatos** à certificação Azure Administrator Associate

---

## Pré-requisitos

Antes de iniciar o curso:

- Conhecimento básico de conceitos de nuvem (AZ-900)
- Familiaridade com sistemas operacionais
- Experiência com redes e virtualização
- Conhecimento básico de PowerShell ou Azure CLI
