## Descricao do Conteudo

In this session, you will explore key concepts such as configuring VNet peering, managing traffic flow with routes, and implementing intersite connectivity.

## Topicos Abordados

- Configuracao de VNet Peering
- Gerenciamento de fluxo de trafego com rotas
- Implementacao de conectividade entre sites (intersite connectivity)
- Administracao de trafego de rede no Azure

## Conceitos-Chave para o Exame

### VNet Peering

- Conexao entre redes virtuais do Azure
- Permite comunicacao privada entre VNets
- Pode ser regional ou global
- Trafego utiliza a rede backbone da Microsoft

### Gerenciamento de Rotas

- User Defined Routes (UDR)
- Route Tables
- Next hop types
- Controle do fluxo de trafego

### Conectividade Intersite

- Conexao entre redes locais e Azure
- VPN Gateway
- ExpressRoute
- Cenarios hibridos
